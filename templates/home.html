Welcome to the Note API!

This is a simple web application for managing notes.

Explore the API to create, update, and manage your notes.

API Endpoints:

Create/Add Note: http://127.0.0.1:8000/api/notes/

To View Note by Id: http://127.0.0.1:8000/api/notes/id_number change id_number to exact number

To Update the Note by Id: http://127.0.0.1:8000/api/notes/id_number/update change id_number to exact number

To Delete the Note by Id: http://127.0.0.1:8000/api/notes/id_number/delete change id_number to exact number

Also Explore the swagger UI to reach out: http://127.0.0.1:8000/swagger/
